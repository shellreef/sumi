SUMI Anonymous File Transfer

SUMI is a new file sharing program that aims for total anonymity for the uploader. It is a work in progress; although anonymous file transfer works there are some rough edges.

Developers: the latest source is now in CVS. Patches welcome!

Official IRC channel is #sumi at irc.liquidirc.com, key anon. LiquidIRC was chosen because it anonymizes the hostmask automatically. Sample files are served here, though you can also chat and ask questions.

SUMI Client GUI Mini-HOWTO

If you want to try out SUMI, this tentative guide may help. If you have any questions or get stuck, feel free to ask for help in the channel or via email.

  1. Download and unpack the latest release to a directory of your choice.
  2. Proceed to SUMI Mini-HOWTO for Windows or SUMI Mini-HOWTO for Unix depending on your OS. Continue below after completing the Windows- or Unix- specific steps.

  3. Welcome back. In your IRC client, type /server irc.liquidirc.com and then /join #sumi anon.
  4. Feel free to say hi. The developers have a ! or @ in front of their name, the servers +.
  5. To download a small sample file, use /sumi get sumi-00 #3. If this works, wait for sumi-00 to list and download the other, larger files if you want to. All the files offered on this bot are believed to be legal to distribute. If you get "No such nick", make sure that sumi-00 is signed on -- sometimes it is not available.
  6. If you're so inclined, try to discover the true source of the files from sumi-00. If SUMI is performing as intended, the source should be anonymous. If not, please contact a developer.
  7. Setup your own SUMI server! For starters, edit sumiserv.cfg to suit your needs (sharing the files you wish to, etc.), and run python sumiserv.py. This step is more involved, if you require any assistance feel free to ask in the channel.
  8. Enjoy anonymous serving!

SUMI Mini-HOWTO for Windows
  1. Install mIRC.
  2. Run the included sumiinst.exe installer.
  3. Make sure you have mIRC integration selected. Proceed with the installation. If you don't have SUMI mIRC already installed, it should popup a dialog saying its about to install, and install itself to your Aliases.ini.
  4. If you installed SUMI to a directory other than C:\Program Files\SUMI, open Aliases.ini (usually in C:\Program Files\mIRC) and edit the run line to match the real location of sumigetw.
  5. Continue here

SUMI Mini-HOWTO for Unix
  1. You need Python. Install it however is normal to do so on your OS.
  2. Install Xchat, at least version 2. Alternatively, install irclib, but xchat2 is preferred.
  3. Edit client-side/xchat.pl so that the $SGW_PATH points to where you placed sumigetw.py.
  4. Start xchat, then go to the menu item X-Chat -> Load Plugin or Script, navigate to sumi/client-side/xchat.pl and load it. Optionally, use cp sumi/client-side/xchat.pl ~/.xchat2/sumi.pl if you want to autoload the SUMI script whenever xchat starts.
  5. xchat should say "SUMI xchat2 transport loading".
  6. Continue here.
