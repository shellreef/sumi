SUMI Anonymous File Transfer

SUMI is a new file sharing program that aims for total anonymity for the uploader. It is a work in progress; although anonymous file transfer works there are some rough edges.

Developers: the latest source is now in CVS. Patches welcome!

Official IRC channel is #sumi at irc.liquidirc.com, key anon. LiquidIRC was chosen because it anonymizes the hostmask automatically, for example this is my whois information:
